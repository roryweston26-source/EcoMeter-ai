What's accurate: token counts (±10–15% English prose), per-token pricing from official sources.

What's estimated: context replay cost (assumes full history resent each turn), system prompt overhead (2–8k tokens per platform), reasoning tokens for o1/o3/R1 (highly variable), image tokens (average per platform).
